(a) Install a Web Server and Publish Website on Intranet Steps: (Using XAMPP – Apache Server) 1. Install Web Server Download XAMPP from https://www.apachefriends.org Install it on your system Open XAMPP Control Panel Start Apache server 2. Create Website Files Create a folder inside: C:\xampp\htdocs\mywebsite Add a file: index.html Example content: My Intranet Website

Welcome to Intranet Website

This site is running on local server (Apache).

3. Run Website on Intranet Open browser Type: http://localhost/mywebsite/ ✔ Website is now accessible within local network (Intranet if shared)
